# Django URL Routing Practice 🚀

This is a beginner-level Django practice project that demonstrates:

- Creating multiple views
- URL routing configuration
- Connecting app-level views with project-level URLs
- Returning HttpResponse with HTML styling

---

project2/

├── app1/

│   ├── apps.py
│
├── project2/
│   ├── settings.py
│   ├── urls.py
│
└── manage.py

---

## 🔗 Implemented Routes

| URL | View Function |
|-----|--------------|
| /gautam/ | Returns simple HttpResponse |
| /ganesh/ | Returns styled HTML heading |
| /chunu/ | Returns colored marquee HTML response |

## ▶️ How to Run the Project

1. Clone the repository
2. Create virtual environment
3. Install Django
4. Run:

```bash
python manage.py runserver
```

5. Open browser:

```
http://127.0.0.1:8000/gautam/
http://127.0.0.1:8000/ganesh/
http://127.0.0.1:8000/chunu/
```
